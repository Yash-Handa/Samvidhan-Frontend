Samvidhan - The API

An open-source public GraphQL API representation of The Constitution of India

Usage

This GraphQL API is build with Apollo Server and is compatible with all GraphQL Clients written in any programming language hence it can be used for building any client application.

Common GraphQL Clients available:

  1. Android / Java:

    1. apollo-android

    2. graphql-java

  2. IOS / Swift:

    1. apollo-ios

  3. Web / JavaScript:

    1. FetchQL

    2. apollo-client

    3. Relay

  4. Other useful stuff (Here)

API EndPoint

All your requests will be send to only one end point:

        http://samvidhan.herokuapp.com/graphql/

Documentation

Since this being a GraphQL API it is self documented and can be completely explained from the PlayGround itself.
