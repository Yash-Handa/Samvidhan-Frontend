Samvidhan - The API

This a an open-source public GraphQL API representation of The Constitution of India

About

This is a fun side-project made as Summer Training Project for 2nd Year Summer Break. This is one of it's kind API which provides the entire Constitution of India as a service. Furthermore it is Completely OpenSource and open for contributions. This API will enable other awesome developers to build great Apps, Websites, Softwares and CLI tools at a very rapid and flexible manner.

Advantages

  1. In memory catching of the entire Indian Constitution using Redis.io data structure RedisJSON.

  2. Supports Automatic Persisted Queries out of the box.

  3. Build on GraphQL, It can support any form or complexity in the Queries.

  4. Supports Pagination for long responses.

  5. Response Catching is done in Apollo Server for faster response generation.

  6. Nginx is used as reverse proxy which reduces response time.

  7. The Project is completely Open-Source and open to contributions.

Open Source ❤️

The entire project is Open Source and is spread across 3 septate Repositories

  1. Samvidhan-Frontend: This repository contains the entire the source code of the this entire web site.
  2. Samvidhan: This repository contains the source code of the Samvidhan GraphQL API.
  3. The_Constitution_Of_India: This repository contains the entire Constitution of India as a single JSON file which acts as the Backend for the API.
